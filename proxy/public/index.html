🚀 Quick Start

1

Get your token

Click "Get My Free Token" above. Your token is saved in your browser automatically.

2

Set environment variables

Add these to your shell or .env file:

# In your terminal or .env file
export XPLLC_PROXY_URL=https://your-deployment-url.com
export XPLLC_USER_TOKEN=your-token-here    # from step 1
3

Clone & run any agent

Clone the repo and run any of the 47 agents:

# Clone the repo
git clone https://github.com/naimmh608-alt/Xpllc-Claw.git
cd Xpllc-Claw

# Code review a file
node scripts/groq-runner.js \
  --proxy $XPLLC_PROXY_URL \
  --token $XPLLC_USER_TOKEN \
  --agent code-reviewer \
  --file src/app.ts

# Auto fix build errors
npm run build 2>&1 | node scripts/groq-runner.js \
  --proxy $XPLLC_PROXY_URL \
  --token $XPLLC_USER_TOKEN \
  --agent build-error-resolver --stdin

# Plan a feature
node scripts/groq-runner.js \
  --proxy $XPLLC_PROXY_URL \
  --token $XPLLC_USER_TOKEN \
  --agent planner \
  --prompt "Add OAuth2 login with Google"
4

Or use the API directly (curl / fetch)

The API is OpenAI-compatible. Drop in any HTTP client:

curl -X POST https://your-deployment.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "X-User-Token: YOUR_TOKEN" \
  -d '{
    "model": "llama-3.1-70b-versatile",
    "messages": [{"role":"user","content":"Review this code..."}],
    "stream": false
  }'
